QuickWeather jest naprawdę szybki. Po prostu otwórz aplikację i zobacz pogodę w tej chwili. Nie ma ekranu ładowania ani czasu oczekiwania. Pogoda jest dostarczana bez dodatkowych śmieci, tylko czyste i jasne informacje.

Możesz dodać tyle lokalizacji do bocznej szuflady, ile chcesz, aby ich pogoda była dostępna za jednym dotknięciem. QuickWeather może nawet określić Twoją bieżącą lokalizację, jeśli na to zezwolisz (informacje o lokalizacji nie są gromadzone). 

QuickWeather jest zawsze darmowy (bez IAP i reklam) i zawsze Open Source Oto, co otrzymasz

<ul><li>Pełnoekranowy radar</li><li>Aktualna pogoda, temperatura, „odczuwalna” temperatura, opady, prędkość wiatru, wilgotność, ciśnienie, punkt rosy, widoczność</li><li>Aktualna i prognozowana temperatura i wykres opadów</li><li>Powiadomienia o bieżącej pogodzie</li><li>Powiadomienia o alertach pogodowych</li><li>Godzinowa temperatura i opady (48 godzin)</li><li>3-godzinna temperatura i Opady (5 dni)</li><li>Dzienna prognoza pogody oraz wysoka i niska temperatura (7 dni)</li><li>Dzienny indeks UV</li>
QuickWeather jest obsługiwany przez <a href="https://openweathermap.org/">OpenWeatherMap.org</a>. QuickWeather wymaga aktywnego klucza API z OpenWeatherMap do działania. Jeśli go nie masz, możesz zarejestrować się w celu uzyskania bezpłatnego klucza API <a href="https://home.openweathermap.org/subscriptions/unauth_subscribe/onecall_30/base">tutaj</a>. Wymagane uprawnienia <ul><li><b>Internet</b> — wymagany do łączenia się z internetem i pobierania informacji o pogodzie</li></ul> Opcjonalne uprawnienia <ul><li><b>Lokalizacja</b> — wymagany do pobrania bieżącej lokalizacji użytkownika, jeśli jest to wymagane.</li><li><b>Lokalizacja w tle</b> — wymagany do pobrania bieżącej lokalizacji użytkownika dla powiadomienia pogodowe i alerty, jeśli są wymagane.</li><li><b>Odbierz ukończone uruchamianie</b> — wymagane do zmiany harmonogramu usługi powiadamiania przy każdym uruchomieniu</li></ul>
